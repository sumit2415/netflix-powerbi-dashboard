# 📺 Netflix Power BI Dashboard

## 📌 Project Overview

This project presents an interactive **Netflix Analytics Dashboard** built using **Microsoft Power BI**. It analyzes Netflix's content library to uncover trends in movies and TV shows based on genres, countries, ratings, release years, and durations.

The dashboard provides meaningful business insights through interactive visualizations and demonstrates practical Power BI skills used in real-world data analytics.

---

## 🎯 Objectives

- Analyze Netflix content distribution.
- Compare Movies and TV Shows.
- Identify top contributing countries.
- Analyze content ratings and genres.
- Visualize release year trends.
- Build an interactive business dashboard.

---

## 🛠️ Tools Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX
