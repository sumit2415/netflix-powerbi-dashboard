# Netflix Power BI Dashboard

## Project Overview
This project analyzes Netflix Movies and TV Shows using Power BI to uncover trends in content, genres, ratings, release years, and country-wise distribution.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## Key Features
- Interactive dashboard
- Genre analysis
- Country-wise content distribution
- Movies vs TV Shows comparison
- Rating distribution
- Release year trends

## Dataset
Netflix Movies and TV Shows Dataset

## Skills Demonstrated
Data Cleaning, Data Visualization, Power Query, DAX, Dashboard Design, Business Insights
